# ASL Alphabet Classification using CNN

## Overview
This project builds a Convolutional Neural Network (CNN) to classify American Sign Language (ASL) alphabet images.

## Dataset
- 29 classes (A–Z + space + delete + nothing)
- Images resized to 64x64

## Model
- Conv2D + MaxPooling layers
- Fully connected layers
- Softmax output

## Results
- Achieved ~79% accuracy

## Technologies
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
